The module `manim.utils.docbuild.module_parsing` is a utility for parsing Manim library modules to extract documentation information. It provides a function `parse_module_attributes()` that reads Python files, generates Abstract Syntax Trees (ASTs), and extracts details about type aliases, module-level attributes, and TypeVar definitions. The function returns three dictionaries: `ALIAS_DOCS_DICT` (type alias documentation), `DATA_DICT` (documented module-level attributes), and `TYPEVAR_DICT` (TypeVar definitions), organized by module names.
