The `manim.mobject.text.text_mobject` module provides text rendering classes in Manim, primarily `Text`, `Paragraph`, and `MarkupText`. These classes enable rendering text using Pango library, supporting various text formatting options like font style, color, weight, gradient, and multi-language text. The module offers flexible text manipulation, character-level styling, and handles complex text rendering with features like ligature management and markup support.
