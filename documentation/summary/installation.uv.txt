Here's a concise summary of the Manim installation process using uv:

1. Install uv (Python package management tool):
   - MacOS/Linux: `curl -LsSf https://astral.sh/uv/install.sh | sh`
   - Windows: `powershell -c "irm https://astral.sh/uv/install.ps1 | iex"`

2. Install Python:
   - Run `uv python install`

3. (Optional) Install LaTeX:
   - Windows: Install MiKTeX
   - MacOS: Install MacTeX
   - Linux: Install TeX Live (e.g., `sudo apt install texlive-full`)

4. Install Manim:
   ```bash
   uv init manimations
   cd manimations
   uv add manim
   ```

5. Verify installation:
   - Run `uv run manim checkhealth`

Alternative global installation:
- `uv tool install manim`

Note: For Linux, additional system dependencies may be required (build-essential, python-dev, cairo-dev, pango-dev).
