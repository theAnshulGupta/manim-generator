Manim v0.17.3 is a release featuring contributions from 35 people, with 42 pull requests merged. Key highlights include: new directed graph (DiGraph) mobject, Scene.replace() method, individualized radius for Polygram.round_corners(), opacity setting for Tex mobjects, and a new StealthTip shape. Performance improvements were made in various areas like Text mobject creation, bezier operations, and OpenGL rendering. Several bugs were fixed, including issues with label_constructor, axis labels, and animation behaviors. Documentation was updated with type hints, clarifications, and link corrections. Minor code quality improvements and dependency updates were also implemented.
