Manim v0.18.0 (released Nov 11, 2023) introduced significant changes, including a complete rewrite of the color system, removal of the colour library dependency, and introduction of ManimColor with hundreds of new predefined colors. Key highlights include a new 'manim checkhealth' CLI command, Binder integration for interactive documentation examples, Python 3.12 support, and automatic LaTeX auxiliary file cleanup. The release featured 3 new rate functions, 2 new mobjects (LabeledLine and LabeledArrow), various enhancements and bug fixes, with contributions from 41 people across 59 merged pull requests.
