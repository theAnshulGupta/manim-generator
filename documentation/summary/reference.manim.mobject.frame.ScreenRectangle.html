`ScreenRectangle` is a Manim class that creates a rectangular shape representing a screen with customizable aspect ratio and height. Key features: - Inherits from `Rectangle` - Default aspect ratio is 16:9 (1.7777...) - Default height is 4 units - Can modify aspect ratio, which automatically adjusts width - Useful for creating screen-like rectangles in animations Example usage: ```python screen = ScreenRectangle(aspect_ratio=16/9, height=5) ``` The class allows easy creation of screen representations with flexible sizing and proportions for Manim animations.
