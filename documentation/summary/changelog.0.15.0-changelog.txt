The Manim Community v0.15.0 release introduced several key changes: breaking changes to arrow tips and bar charts, deprecated methods removal, new features like media embedding in Jupyter and more control over Scene.wait(), enhancements to graph layouts and text inheritance, bug fixes for animations and rendering, improved documentation, and various code quality improvements. Notable updates include better OpenGL compatibility, performance optimizations, and refactoring of several modules. 34 contributors participated, with 71 pull requests merged, focusing on improving library functionality, performance, and user experience.
