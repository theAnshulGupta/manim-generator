The `logger_utils` module provides utilities for creating and configuring Manim's logging system. It allows accessing the logger via `manim.logger` or `logging.getLogger("manim")`, and provides a `console` object for screen messages. The module includes functions like `make_logger()` to create loggers with specified verbosity, `parse_theme()` to configure rich text styling, and `set_file_logger()` to add file handlers. It also contains a `JSONFormatter` class for custom JSON log formatting, utilizing the `rich` library for enhanced text output.
