Manim installation guide using uv, a Python package management tool. First, install uv via a terminal command for your OS. Then, install Python using `uv python install`. Optional LaTeX installation is recommended for rendering mathematical formulas. Installation steps vary by OS: for MacOS/Windows, create a project directory and add Manim with `uv init manimations` and `uv add manim`. Linux requires additional system dependencies (C compiler, Python dev headers, pkg-config, Pango, Cairo) before similar project setup. Verify installation with `uv run manim checkhealth`. Alternative global installation possible using `uv tool install manim`. Python version can be specified during project initialization using `uv init --python 3.x manimations`.
