Summary of `three_dimensions` module in Manim: 3D Mobject Classes: - `Arrow3D`: 3D arrow with cylindrical line and conical tip - `Cone`: Circular cone - `Cube`: 3D cube - `Cylinder`: Defined by height, radius, and direction - `Dot3D`: Spherical dot - `Line3D`: Cylindrical line for 3D scenes - `Prism`: Rectangular cuboid - `Sphere`: 3D sphere - `Surface`: Parametric surface with checkerboard pattern - `Torus`: 3D ring-shaped object These classes allow creating and manipulating three-dimensional geometric shapes in Manim animations.
