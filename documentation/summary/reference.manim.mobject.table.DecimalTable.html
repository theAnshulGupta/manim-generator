Summary of DecimalTable: DecimalTable is a specialized Table class for displaying decimal numbers with the following key features: 1. Purpose: - Creates tables with decimal number entries - Automatically converts table entries to DecimalNumber mobjects 2. Key Parameters: - `table`: 2D array of numbers/strings - `element_to_mobject`: Defaults to DecimalNumber - `element_to_mobject_config`: Controls decimal place formatting (default: 1 decimal place) 3. Example Usage: ```python x_vals = [-2,-1,0,1,2] y_vals = np.exp(x_vals) t0 = DecimalTable( [x_vals, y_vals], row_labels=[MathTex("x"), MathTex("f(x)=e^{x}")], h_buff=1, element_to_mobject_config={"num_decimal_places": 2} ) ``` 4. Key Behavior: - Automatically rounds/truncates decimal places - Converts input values to formatted decimal number mobjects - Inherits all Table class functionality
