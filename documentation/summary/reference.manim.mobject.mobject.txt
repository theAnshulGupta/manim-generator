This documentation page describes the core `Mobject` (Mathematical Object) class in Manim, which is the base class for all objects that can be displayed on screen. 

Key Points:
- Base class for all visual elements in Manim
- Supports various updaters and animation methods

Type Aliases:
- `TimeBasedUpdater`: A callable function that updates a Mobject over time
- `NonTimeBasedUpdater`: A callable function that updates a Mobject
- `Updater`: Can be either time-based or non-time-based

Main Classes:
- `Group`: Groups multiple Mobjects together
- `Mobject`: The fundamental base class for visual objects

Notable Function:
- `override_animate()`: A decorator that allows custom animation for specific methods

Example Code:
```python
class CircleWithContent(VGroup):
    def __init__(self, content):
        super().__init__()
        self.circle = Circle()
        self.content = content
        self.add(self.circle, content)
        content.move_to(self.circle.get_center())

    def clear_content(self):
        self.remove(self.content)
        self.content = None

    @override_animate(clear_content)
    def _clear_content_animation(self, anim_args=None):
        anim = Uncreate(self.content, **anim_args)
        self.clear_content()
        return anim
```

The example demonstrates creating a custom Mobject with a special animation for clearing its content.
