ArrowSquareTip is a square-shaped arrow tip in Manim used when creating arrows. Key details:

Parameters:
- `fill_opacity`: Transparency of tip (default 0)
- `stroke_width`: Width of tip outline (default 3)
- `length`: Size of tip (default 0.35)

Example usage:
```python
Arrow(start_point, end_point, tip_class=ArrowSquareTip)
```

This creates arrows with square-shaped tips instead of the default triangular tips. The tip can be customized via its parameters to adjust size, opacity, and outline width.
