This module provides the `TexTemplate` class for creating and customizing LaTeX templates in Manim. It allows users to define TeX compilation settings like compiler, document class, preamble, and placeholder text. Key methods include `from_file()` for loading templates from files, `add_to_preamble()` and `add_to_document()` for modifying templates, and `get_texcode_for_expression()` for inserting expressions into templates. The class supports deep copying and provides flexibility in creating LaTeX rendering configurations.
