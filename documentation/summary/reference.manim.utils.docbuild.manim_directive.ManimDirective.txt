This is a technical reference for the ManimDirective class used in Manim's documentation build system. It's a specialized directive for rendering Manim animations while building documentation.

Key Characteristics:
- Inherits from Directive base class
- Used to render videos during documentation generation
- Supports various options for controlling rendering

Options Include:
- hide_source: Hide source code
- no_autoplay: Disable video autoplay
- quality: Set rendering quality
- save_as_gif: Save output as GIF
- save_last_frame: Save only the final frame
- ref_classes/functions/methods/modules: Reference tracking

Configuration:
- Requires 1 argument
- Can have content
- Allows whitespace in final argument

Not relevant for typical Manim usage, more of an internal documentation tool.
