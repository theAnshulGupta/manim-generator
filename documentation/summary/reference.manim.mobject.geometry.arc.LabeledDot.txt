LabeledDot is a Manim class that creates a dot with a label in its center. Key details:

Parameters:
- `label`: String or text object to display inside the dot
- `radius`: Optional dot size (auto-calculated if not specified)
- Supports various text types (MathTex, Text, Tex)

Example usage:
```python
dot1 = LabeledDot("42")  # Simple string label
dot2 = LabeledDot(MathTex("a", color=GREEN))  # Colored math label
dot3 = LabeledDot(Text("ii", color=BLUE))  # Colored text label
```

Full example demonstrates creating multiple labeled dots around a square:
```python
class SeveralLabeledDots(Scene):
    def construct(self):
        sq = Square(fill_color=RED, fill_opacity=1)
        self.add(sq)
        dot1 = LabeledDot(Tex("42", color=RED))
        dot2 = LabeledDot(MathTex("a", color=GREEN))
        dot3 = LabeledDot(Text("ii", color=BLUE))
        dot4 = LabeledDot("3")
        dot1.next_to(sq, UL)
        dot2.next_to(sq, UR)
        dot3.next_to(sq, DL)
        dot4.next_to(sq, DR)
        self.add(dot1, dot2, dot3, dot4)
```
