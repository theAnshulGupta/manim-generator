`IntegerTable` is a specialized table class in Manim specifically designed for creating tables with integer values:

Key Features:
- Subclass of `Table`
- Automatically converts table entries to integers
- Rounds decimal entries to nearest integer
- Default cell representation uses the `Integer` class

Example Usage:
```python
t0 = IntegerTable(
    [[0,30,45,60,90],   # Table data
     [90,60,45,30,0]],  
    col_labels=[  # Optional column labels
        MathTex(r"\frac{\sqrt{0}}{2}"),
        MathTex(r"\frac{\sqrt{1}}{2}"),
        ...
    ],
    row_labels=[  # Optional row labels
        MathTex(r"\sin"), 
        MathTex(r"\cos")
    ],
    h_buff=1,  # Horizontal buffer between cells
    element_to_mobject_config={"unit": r"^{\circ}"}  # Additional configuration
)
```

Main Parameters:
- `table`: 2D list of numbers/strings to populate table
- `element_to_mobject`: Mobject class for table entries (default `Integer`)
- Additional Table configuration options can be passed via `**kwargs`

Automatically handles converting and rounding numeric inputs to integers.
