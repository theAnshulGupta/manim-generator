Summary of tex_mobject in Manim:

Tex_mobject provides classes for rendering text using LaTeX in Manim animations:

Key Classes:
1. Tex: Renders LaTeX text in normal mode
2. MathTex: Renders LaTeX text in math mode
3. SingleStringMathTex: Basic LaTeX text rendering
4. BulletedList: Creates a bulleted list
5. Title: Creates an underlined title

Important Note: Users should refer to the "Text With LaTeX" tutorial for detailed usage guidance.

Example Usage:
```python
from manim import Tex, MathTex

# Normal text rendering
text = Tex("Hello LaTeX")

# Math mode rendering
math_text = MathTex("E = mc^2")
```

The module allows seamless integration of LaTeX-rendered text into Manim animations.
