Module for rendering LaTeX text in Manim. Contains classes like `SingleStringMathTex` (base class for rendering LaTeX), `MathTex` (for math mode LaTeX), `Tex` (for normal mode LaTeX), `BulletedList` (for creating bulleted lists), and `Title` (for creating underlined titles). Key features include rendering LaTeX strings, handling color mappings, isolating substrings, and organizing submobjects. The classes support flexible text rendering with various customization options like font size, color, and environment settings.
