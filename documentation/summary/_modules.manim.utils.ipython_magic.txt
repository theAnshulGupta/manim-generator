This module provides an IPython magic command (`%manim`) for rendering Manim scenes directly in Jupyter notebooks. Here's a concise summary:

Key Features:
- Use `%manim` or `%%manim` to render Manim scenes in notebooks
- Supports line and cell magic modes
- Automatically displays rendered images/videos
- Configurable via Manim's configuration options

Basic Usage:
```python
# Line mode
%manim MySceneClass

# Cell mode
%%manim MySceneClass
class MySceneClass(Scene):
    def construct(self):
        # Scene code here
```

Configuration Options:
- `config.media_width`: Control video display width (default: `25vw`)
- `config.media_embed`: Embed media in notebook
- `config.progress_bar`: Hide progress bar with `None`

Example:
```python
%%manim -v WARNING --disable_caching -qm BannerExample

config.media_width = "75%"
config.media_embed = True

class BannerExample(Scene):
    def construct(self):
        banner = ManimBanner().scale(0.7)
        self.play(banner.create())
```

Note: Works best in JupyterLab, with some limitations in classic Jupyter notebooks.
