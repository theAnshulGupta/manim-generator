Module parsing utility for Manim documentation that extracts and organizes module-level type information. The `parse_module_attributes()` function reads Manim files, generates Abstract Syntax Trees, and returns three key dictionaries: `ALIAS_DOCS_DICT` (type alias documentation), `DATA_DICT` (documented module-level attributes), and `TYPEVAR_DICT` (TypeVar definitions). Provides structured data types like `AliasInfo`, `AliasCategoryDict`, and `ModuleLevelAliasDict` to categorize and store type alias information across Manim modules.
