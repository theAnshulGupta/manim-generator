Here's a concise summary of Manim's Output Settings: Rendering Outputs: - Manim generates files in a `media` directory with specific folder structure - Output includes videos, partial movie files, and other support files - Supports different quality levels: - `-ql`: Low (480p, 15 FPS) - `-qm`: Medium (720p, 30 FPS) - `-qh`: High (1080p, 60 FPS) - `-qp`: 2K (1440p, 60 FPS) - `-qk`: 4K (2160p, 60 FPS) Key Command-Line Flags: - `-p`: Play video after rendering - `-f`: Open file browser to output location - `-s`: Save only the last frame - `--save_sections`: Create separate videos for each section - `--format gif`: Render as GIF instead of MP4 Sections: - Use `self.next_section()` to create video sections - Each section can be a separate output video - Can skip animations in a section with `skip_animations=True` Example Folder Structure: ``` project/ └─media ├─videos │ └─scene │ ├─480p15 │ │ └─SquareToCircle.mp4 │ └─1080p60 │ └─SquareToCircle.mp4 └─images ```
