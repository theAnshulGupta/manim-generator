`AnnotationDot` is a specialized `Dot` class in Manim designed for scene annotations. Key characteristics:

- Slightly larger default radius (0.104)
- Bolder stroke width (5)
- Default white stroke color
- Default fill color of a light blue (#58C4DD)

Purpose: Creating prominent dots to highlight or annotate specific points in Manim animations.

Example usage:
```python
dot = AnnotationDot()  # Uses default parameters
custom_dot = AnnotationDot(
    radius=0.2, 
    stroke_width=3, 
    stroke_color=WHITE, 
    fill_color=RED
)
```

Inherits standard `Dot` methods and attributes, with customized visual appearance for annotation purposes.
