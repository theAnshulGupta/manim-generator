The `zoomed_scene` module in Manim provides a specialized scene class `ZoomedScene` for creating animations with zoomed-in views of specific parts of a scene. Key features include: 1. Zooming Functionality: - Allows zooming into a specific section of the scene - Can activate zooming with `self.activate_zooming()` - Supports customizing zoom parameters like zoom factor, display height/width 2. Example Usage: ```python class UseZoomedScene(ZoomedScene): def construct(self): dot = Dot().set_color(GREEN) self.add(dot) self.activate_zooming(animate=False) self.play(dot.animate.shift(LEFT)) class ChangingZoomScale(ZoomedScene): def __init__(self, **kwargs): ZoomedScene.__init__( self, zoom_factor=0.3, zoomed_display_height=1, zoomed_display_width=3, # Additional configuration options ) def construct(self): # Similar zooming operations self.play(self.zoomed_camera.frame.animate.scale(4)) ``` 3. Configurable Parameters: - Zoom factor - Zoomed display dimensions - Frame stroke width - Camera configuration The module provides flexible zooming capabilities for creating detailed, focused animations.
