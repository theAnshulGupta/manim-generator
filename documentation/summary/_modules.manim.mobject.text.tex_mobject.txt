The `manim.mobject.text.tex_mobject` module provides classes for rendering LaTeX text in Manim:

1. `SingleStringMathTex`: Basic building block for rendering single LaTeX strings
   - Converts LaTeX to SVG
   - Handles special cases in LaTeX string parsing
   - Supports font size scaling

2. `MathTex`: LaTeX rendering in math mode
   - Can handle multiple LaTeX strings
   - Supports substring isolation
   - Color mapping for different parts of the text
   - Methods like `get_parts_by_tex()`, `set_color_by_tex()`

3. `Tex`: LaTeX rendering in text mode
   - Similar to `MathTex`, but defaults to text environment

4. `BulletedList`: Creates a list with bullet points
   - Supports custom styling and color per item
   - `fade_all_but()` method to highlight specific items

5. `Title`: Creates an underlined title
   - Optional underline with configurable width
   - Automatically positioned at the top of the scene

Example:
```python
formula = MathTex(r"\int_a^b f'(x) dx = f(b) - f(a)")
list = BulletedList("Item 1", "Item 2", "Item 3")
title = Title("My Animation")
```
