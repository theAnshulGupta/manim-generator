`ImageMobject` is a Manim class for displaying images from files or numpy arrays. Key features:

Parameters:
- `filename_or_array`: Image source (file path or numpy array)
- `scale_to_resolution`: Sets image resolution (default 1080)
- `invert`: Option to invert image colors
- `image_mode`: Color mode (default 'RGBA')

Main methods:
- `fade()`: Sets image opacity
- `set_opacity()`: Controls image transparency
- `set_color()`: Modifies image color
- `get_pixel_array()`: Retrieves pixel data

Example usage:
```python
# Create image from numpy array
image = ImageMobject(np.uint8([[0, 100, 30, 200],
                               [255, 0, 5, 33]]))
image.height = 7
self.add(image)

# Change interpolation style
image.set_resampling_algorithm(RESAMPLING_ALGORITHMS["linear"])
```

Key capabilities:
- Can create images from files or arrays
- Flexible opacity and color manipulation
- Multiple image scaling/interpolation algorithms
