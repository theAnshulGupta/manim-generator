BackgroundRectangle is a Manim class used to create a rectangular background for other mobjects. Key features: Purpose: - Creates a rectangular background behind other objects - Defaults to scene's background color - Customizable opacity and style Parameters: - mobjects: Objects to create background for - color: Background rectangle color (optional) - fill_opacity: Transparency level (default 0.75) - stroke_width: Border width (default 0) - buff: Extra padding around objects Example Code: ```python class ExampleBackgroundRectangle(Scene): def construct(self): circle = Circle().shift(LEFT) triangle = Triangle().shift(2 * RIGHT) # Create background rectangles for objects bg1 = BackgroundRectangle(circle, color=WHITE, fill_opacity=0.15) bg2 = BackgroundRectangle(triangle, color=WHITE, fill_opacity=0.15) # Add backgrounds and objects self.add(bg1, bg2, circle, triangle) # Animate background rectangles self.play(Rotate(bg1, PI/4), Rotate(bg2, PI/2)) ``` Use Case: Highlighting or providing visual separation for objects in a scene.
