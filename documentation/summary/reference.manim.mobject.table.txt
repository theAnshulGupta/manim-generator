Summary of Manim Table Documentation:

The `Table` module in Manim provides multiple ways to create tables with different types of entries:

1. Basic Table Types:
- `Table`: Generic table with flexible content
- `DecimalTable`: For decimal numbers
- `IntegerTable`: For integer values
- `MathTable`: For LaTeX entries
- `MobjectTable`: For custom Mobject entries

2. Key Features:
- Can add row/column labels
- Support for highlighting cells
- Option to include/exclude table lines
- Customizable styling and positioning

3. Example Demonstrates:
- Creating tables with text, math, integers
- Highlighting specific cells
- Adding lines and custom colors
- Creating varied table layouts

4. Basic Usage Example:
```python
table = Table(
    [["First", "Second"],
     ["Third", "Fourth"]],
    row_labels=[Text("R1"), Text("R2")],
    col_labels=[Text("C1"), Text("C2")],
    top_left_entry=Text("TOP")
)
table.add_highlighted_cell((2,2), color=GREEN)
```

Each table type offers specific customization for different data presentation needs.
