The `rotation` module in Manim provides two key animations for rotating mobjects:

1. `Rotate`: An animation that rotates a mobject
   - Key parameters:
     - Specify angle of rotation
     - Rotation axis
     - Optional pivot point
   - Example:
     ```python
     circle.animate.rotate(PI/2)  # Rotates 90 degrees
     ```

2. `Rotating`: Likely a continuous rotation animation
   - Specifics would require more detailed documentation

These animations allow you to create rotational movements for geometric objects in Manim animations.
