Docker Installation and Usage for Manim:

Docker Images:
- Tags available: 
  - latest: most recent version
  - stable: latest released version
  - vX.Y.Z: specific released versions

Basic Docker Usage:
```bash
# Render a scene from a file
docker run --rm -it -v "/full/path/to/your/directory:/manim" manimcommunity/manim manim -qm test_scenes.py CircleToSquare

# For Linux users (prevent permission issues)
docker run --rm -it --user="$(id -u):$(id -g)" -v "/full/path/to/your/directory:/manim" manimcommunity/manim manim -qm test_scenes.py CircleToSquare
```

Persistent Container:
```bash
# Create a named container
docker run -it --name my-manim-container -v "/full/path/to/your/directory:/manim" manimcommunity/manim bash

# Start container in background
docker start my-manim-container

# Render scene in persistent container
docker exec -it my-manim-container manim -qm test_scenes.py CircleToSquare
```

JupyterLab:
```bash
# Run JupyterLab 
docker run -it -p 8888:8888 manimcommunity/manim jupyter lab --ip=0.0.0.0
```

Note: Some CLI flags like -p and -f are not supported in Docker.
