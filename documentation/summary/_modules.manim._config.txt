Summary of manim._config:

1. Core Functionality:
- Sets up global configuration, logging, and console output for Manim
- Provides a centralized configuration management system

2. Key Components:
- `config`: Global configuration object (ManimConfig)
- `logger`: Logging utility
- `console` and `error_console`: Output streams
- `cli_ctx_settings`: CLI context settings

3. `tempconfig` Context Manager:
- Temporarily modifies global configuration
- Allows temporary changes without permanent modification
- Restores original configuration after use

Example:
```python
with tempconfig({"frame_height": 100.0}):
    print(config["frame_height"])  # 100.0
# Original config restored afterwards
```

4. Additional Notes:
- Suppresses verbose logging from PIL and matplotlib
- Provides a clean interface for configuration management
