`TangentLine` is a Manim class for creating lines tangent to a VMobject (like a circle or curve). Key parameters: - `vmob`: The VMobject to draw the tangent on - `alpha`: Position along the shape (0-1) - `length`: Length of the tangent line - `d_alpha`: Small delta for calculating tangent direction Example usage: ```python class TangentLineExample(Scene): def construct(self): circle = Circle(radius=2) # Tangent line at right side of circle line_1 = TangentLine(circle, alpha=0.0, length=4, color=BLUE_D) # Tangent line at top-left of circle line_2 = TangentLine(circle, alpha=0.4, length=4, color=GREEN) self.add(circle, line_1, line_2) ``` This creates tangent lines of length 4 at different points on a circle, demonstrating how to specify the tangent's location and properties.
