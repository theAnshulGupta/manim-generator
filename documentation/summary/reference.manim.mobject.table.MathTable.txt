`MathTable` is a specialized table class in Manim designed for creating LaTeX-based mathematical tables:

Key Features:
- Subclass of `Table`
- Automatically converts table entries to LaTeX math using `MathTex`
- Renders mathematical expressions in table cells

Example Usage:
```python
class MathTableExample(Scene):
    def construct(self):
        table = MathTable(
            [["+", 0, 5, 10],
             [0, 0, 5, 10],
             [2, 2, 7, 12],
             [4, 4, 9, 14]],
            include_outer_lines=True
        )
        self.add(table)
```

Parameters:
- `table`: 2D array of mathematical content
- `element_to_mobject`: Defaults to `MathTex`
- Supports additional `Table` arguments via `kwargs`

Main purpose is to easily create mathematically formatted tables with LaTeX rendering for each cell.
