The `tex_templates` module provides LaTeX template utilities for Manim:

1. `TexFontTemplates`: A collection of TeX templates for various fonts, sourced from a specific font showcase website.

2. `TexTemplateLibrary`: A collection of basic TeX template objects.

These classes help users customize LaTeX rendering in Manim by providing pre-defined templates for different font styles and basic LaTeX configurations. They can be used when creating mathematical text or formulas to control the appearance and styling of the rendered LaTeX.
