The `code_mobject` module in Manim provides the `Code` class for creating highlighted source code listings with customizable parameters:

Key Features:
- Renders source code with syntax highlighting
- Supports multiple programming languages
- Configurable code styling and display options

Example Usage:
```python
from manim import Code

# Basic code display
code = Code(
    code='print("Hello, World!")', 
    language='python',
    line_spacing=0.5,
    font_size=24
)
```

Main Configuration Options:
- `code`: The source code string
- `language`: Programming language for syntax highlighting
- `line_spacing`: Space between code lines
- `font_size`: Text size
- `tab_width`: Indentation width
- `background_stroke_width`: Border around code block
- `style`: Syntax highlighting theme

The `Code` class allows easy insertion of programmatically styled code snippets into Manim animations.
