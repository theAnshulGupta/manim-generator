This module provides utilities for configuring Manim's logging system:

1. Key Functions:
- `make_logger()`: Creates a logger and consoles with customizable settings
- `parse_theme()`: Configures rich text styling for console output
- `set_file_logger()`: Adds file logging for scenes

2. Features:
- Uses Rich library for enhanced console output
- Supports custom color themes
- Configurable logging levels and timestamps
- Generates JSON-formatted log files
- Highlights specific keywords in log messages

3. Main Components:
- Logger can be accessed via `manim.logger`
- Two consoles: one for stdout, one for stderr
- Supports configuration via config parser
- Customizable log verbosity and appearance

The module is primarily used internally for configuring Manim's logging system and is not typically directly used by end users.
