Summary:

The `manim_directive` is a Sphinx documentation tool for embedding Manim animations directly in documentation. Key features include:

Basic Usage:
```rst
.. manim:: MyScene

    class MyScene(Scene):
        def construct(self):
            ...
```

Supported Options:
- `hide_source`: Hide source code
- `no_autoplay`: Prevent video autoplay
- `quality`: Set render quality ('low', 'medium', 'high', 'fourk')
- `save_as_gif`: Render as GIF
- `save_last_frame`: Display only last frame
- `ref_classes`: List reference classes
- `ref_functions`: List reference functions
- `ref_methods`: List reference methods

Example with options:
```rst
.. manim:: MyScene
    :hide_source:
    :quality: high
```

Supports rendering scenes from doctests and allows flexible documentation integration with Manim animations.
