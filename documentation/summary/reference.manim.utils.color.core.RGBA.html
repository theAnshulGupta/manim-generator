RGBA is a color representation method in Manim with multiple methods for color manipulation: Key Methods: - `from_hex()`: Create color from hex string - `from_rgb()`: Create color from RGB array - `from_rgba()`: Create color from RGBA array - `to_hex()`: Convert color to hex - `to_rgb()`: Convert color to RGB array - `to_rgba()`: Convert color to RGBA array Color Transformation Methods: - `lighter()`: Make color lighter - `darker()`: Make color darker - `invert()`: Linearly invert color - `interpolate()`: Blend between two colors - `opacity()`: Set color's transparency - `contrasting()`: Get contrasting color (light/dark) Color Space Conversion Methods: - `into()`: Convert to different color space - `from_hsl()`: Create from HSL array - `from_hsv()`: Create from HSV array - `to_hsl()`: Convert to HSL array - `to_hsv()`: Convert to HSV array The class provides flexible color manipulation and conversion utilities.
