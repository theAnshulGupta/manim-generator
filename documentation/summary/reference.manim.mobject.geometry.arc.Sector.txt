Sector is a Manim class for creating a wedge-shaped section of a circle. Here's a concise summary:

Key Features:
- Inherits from AnnularSector
- Creates a pie slice/sector of a circle
- Customizable radius and angle

Basic Usage Example:
```python
from manim import *

class ExampleSector(Scene):
    def construct(self):
        # Default sector (full quarter circle)
        sector1 = Sector(radius=2)
        
        # Custom sector with specific angle
        sector2 = Sector(radius=2.5, angle=60*DEGREES).move_to([-3, 0, 0])
        
        sector1.set_color(RED)
        sector2.set_color(PINK)
        
        self.add(sector1, sector2)
```

Parameters:
- `radius`: Size of the sector (default=1)
- `angle`: Angular width of sector (controlled via kwargs)
- Other standard Manim mobject styling options available

The class allows easy creation of circular sectors with flexible customization options.
