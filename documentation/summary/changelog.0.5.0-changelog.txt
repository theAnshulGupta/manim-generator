Manim v0.5.0 introduced significant updates including a new OpenGL renderer with a --use_opengl_renderer flag, deprecated ShowCreation in favor of Create, added new animations like FadeTransform and TransformMatchingShapes, introduced Dot3D and Angle/RightAngle mobjects, and improved rendering capabilities. The release featured 64 merged pull requests from 35 contributors, with enhancements to rendering quality, animation flexibility, and overall library infrastructure. Key improvements included better support for parametric functions, animation caching, improved play() logic, and numerous bug fixes across various components of the library.
