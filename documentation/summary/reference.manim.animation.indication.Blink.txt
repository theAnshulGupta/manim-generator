Summary of Blink Animation in Manim:

Blink is an animation that makes a Mobject (like text or shape) repeatedly appear and disappear, simulating a blinking effect.

Key Parameters:
- `mobject`: Object to blink
- `time_on`: Duration object is visible (default 0.5s)
- `time_off`: Duration object is hidden (default 0.5s)
- `blinks`: Number of blink cycles (default 1)
- `hide_at_end`: Whether object remains hidden after animation (default False)

Example Usage:
```python
class BlinkingExample(Scene):
    def construct(self):
        text = Text("Blinking").scale(1.5)
        self.add(text)
        self.play(Blink(text, blinks=3))  # Blinks text 3 times
```

This creates a text object that blinks 3 times during the animation.
