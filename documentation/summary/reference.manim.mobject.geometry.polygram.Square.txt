Square is a Manim geometry class representing a rectangle with equal side lengths. Here's a concise summary:

Class Details:
- Inherits from Rectangle
- Default side length is 2.0
- Can be customized with various parameters

Key Usage Example:
```python
class SquareExample(Scene):
    def construct(self):
        # Create squares of different sizes
        square_1 = Square(side_length=2.0).shift(DOWN)
        square_2 = Square(side_length=1.0).next_to(square_1, UP)
        square_3 = Square(side_length=0.5).next_to(square_2, UP)
        self.add(square_1, square_2, square_3)
```

Main Parameters:
- `side_length`: Controls the size of the square (default 2.0)
- Supports additional Rectangle configuration options via `**kwargs`

Key Methods/Attributes:
- `animate`: Animate square transformations
- `shift()`: Move square
- `next_to()`: Position relative to other objects
- `side_length`: Access current side length
- `width`, `height`: Dimensional properties
