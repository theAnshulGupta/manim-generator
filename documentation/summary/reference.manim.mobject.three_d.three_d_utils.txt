This module provides utility functions for 3D vector mobjects, specifically for getting corner points, indices, and unit normals. Key functions include:

1. `get_3d_vmob_start_corner()`: Returns the start corner point of a 3D vector mobject
2. `get_3d_vmob_end_corner()`: Returns the end corner point of a 3D vector mobject
3. `get_3d_vmob_start_corner_index()`: Returns the start corner index (always 0)
4. `get_3d_vmob_end_corner_index()`: Returns the end corner index
5. `get_3d_vmob_start_corner_unit_normal()`: Returns the unit normal vector at the start corner
6. `get_3d_vmob_end_corner_unit_normal()`: Returns the unit normal vector at the end corner
7. `get_3d_vmob_unit_normal()`: Returns a unit normal vector for a specific point index
8. `get_3d_vmob_gradient_start_and_end_points()`: Returns a tuple of start and end points for gradients

These functions are primarily used for advanced 3D vector mobject manipulations in Manim.
