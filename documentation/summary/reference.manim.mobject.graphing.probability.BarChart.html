BarChart is a Manim class for creating bar charts with customizable features: Key Parameters: - `values`: Sequence of numeric values determining bar heights - `bar_names`: Optional names for each bar - `y_range`: Y-axis range (auto-calculated if not specified) - `bar_colors`: Colors for bars - `bar_width`: Width of bars (0-1) Main Methods: 1. `change_bar_values()`: Update bar heights dynamically 2. `get_bar_labels()`: Add numeric labels to bars Example: ```python chart = BarChart( values=[-5, 40, -10, 20, -3], bar_names=["one", "two", "three", "four", "five"], y_range=[-20, 50, 10], y_length=6 ) bar_labels = chart.get_bar_labels() ``` Key Features: - Supports positive and negative values - Automatically generates colors - Flexible axis and label configuration - Can dynamically update bar values
