ParametricFunction allows you to plot parametric curves in Manim by defining a function that maps a parameter t to (x, y, z) coordinates. Key features include: Parameters: - `function`: Lambda/function defining curve coordinates (x(t), y(t), z(t)) - `t_range`: Range of t values to plot (default [0, 1]) - `use_smoothing`: Interpolates between points (default True) - `discontinuities`: Specify points where function is undefined Examples: ```python # 2D parametric curve func = ParametricFunction( lambda t: (np.sin(2*t), np.sin(3*t), 0), t_range=(0, TAU) ).set_color(RED) # 3D parametric spring curve = ParametricFunction( lambda u: ( 1.2 * np.cos(u), 1.2 * np.sin(u), u * 0.05 ), color=RED, t_range=(-3*TAU, 5*TAU) ) ``` Special notes: - For functions with discontinuities, manually specify points using `discontinuities` parameter - Can be used in both 2D and 3D scenes - Supports various customization options like color, scaling, and smoothing
