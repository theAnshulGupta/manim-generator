Base Mobject Summary: The Mobject (Mathematical Object) is the fundamental class in Manim for creating visual elements with the following key characteristics: Core Attributes: - submobjects: List of contained objects - points: Numpy array of object's points - color: Object's color - width/height/depth: Dimensional measurements Key Methods: 1. Positioning/Transformation - move_to(): Move object to specific point - shift(): Move object by vector - scale(): Resize object - rotate(): Rotate object - next_to(): Position relative to another object 2. Manipulation - add(): Add submobjects - copy(): Create identical copy - become(): Transform into another object - align_to(): Align with another object 3. Updaters - add_updater(): Attach dynamic update function - clear_updaters(): Remove all updaters - update(): Apply all updaters 4. Styling - set_color(): Change color - set_opacity(): Adjust transparency - animate: Create animated transformations 5. Positioning Helpers - to_corner(): Move to screen corner - to_edge(): Move to screen edge - center(): Center on screen Primary Use: Base class for creating and manipulating geometric and text objects in animations. Example workflow: ```python circle = Circle().set_color(RED) # Create red circle circle.shift(RIGHT) # Move right circle.scale(2) # Double size ```
