ZoomedScene is a specialized Scene class that enables zooming into specific parts of a scene for detailed visualization. It allows users to activate zooming, dynamically adjust zoom factors, display dimensions, and frame properties. The provided examples demonstrate how to create a zoomed view of objects like dots and circles, with options to animate zooming, change zoom scale, and modify the zoomed camera frame. Users can customize zoom parameters such as zoom factor, display height/width, and frame stroke width during scene initialization.
