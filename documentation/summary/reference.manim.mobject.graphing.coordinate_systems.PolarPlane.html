PolarPlane is a Manim class for creating polar coordinate systems with configurable parameters: Key Parameters: - `radius_max`: Maximum radius value (default 4.0) - `size`: Diameter of the plane - `radius_step`: Distance between radius lines - `azimuth_units`: Labeling system for angles (e.g. "PI radians", "degrees") - `azimuth_direction`: Direction of angles ("CW" or "CCW") Basic Example: ```python class PolarPlaneExample(Scene): def construct(self): polarplane = PolarPlane( azimuth_units="PI radians", size=6, azimuth_label_font_size=33.6 ).add_coordinates() self.add(polarplane) ``` Key Methods: - `add_coordinates()`: Adds coordinate labels - `get_coordinate_labels()`: Retrieves coordinate labels - `get_axes()`: Returns the axes of the polar plane The class allows extensive customization of polar coordinate visualization, including angle units, label styles, and coordinate display.
