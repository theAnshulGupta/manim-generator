Summary of the `Variable` class in Manim: Purpose: Creates a dynamic text object that displays a label and a value that can be updated in real-time. Key Parameters: - `var`: Initial numeric value - `label`: Label for the variable (text/math) - `var_type`: Display type (DecimalNumber or Integer, default is DecimalNumber) - `num_decimal_places`: Number of decimal places to display Main Attributes: - `label`: The label text - `tracker`: ValueTracker for updating the value - `value`: The numeric value display Basic Usage Example: ```python # Decimal variable var = Variable(0.5, Text("var"), num_decimal_places=3) # Integer variable int_var = Variable(0, Text("int_var"), var_type=Integer) # Updating value var_tracker = on_screen_var.tracker self.play(var_tracker.animate.set_value(10.5)) ``` Advanced Features: - Customize label and value colors - Support for math mode labels - Dynamic value updates using trackers - Can create dependent variables (e.g., x and xÂ²)
