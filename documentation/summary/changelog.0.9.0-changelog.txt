The Manim Community v0.9.0 release, made on August 2, 2021, introduced several key improvements: a new Table mobject, enhanced NumberLine and Axes customization, added Broadcast animation, Circle.from_three_points() method, ParametricSurface color gradients, and OpenGL renderer updates. The release deprecated several older classes and functions, fixed numerous bugs, improved documentation, and refactored testing systems. 35 contributors participated, with 55 pull requests merged, focusing on enhancing library functionality, performance, and user experience across various mobjects and rendering capabilities.
