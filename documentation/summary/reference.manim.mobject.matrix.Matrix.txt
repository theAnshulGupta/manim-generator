The Matrix class in Manim is a VMobject that displays a matrix on the screen. It allows creating matrices with customizable parameters like vertical/horizontal buffer, bracket style, entry alignment, and background options. Key features include:

- Initializes with a 2D array or list of lists
- Configurable entry spacing, bracket style, and alignment
- Optional background rectangles for entries or entire matrix
- Methods to manipulate matrix elements:
  - get_columns(), get_rows(): Return matrix columns/rows
  - get_entries(): Get individual matrix entries
  - set_column_colors(), set_row_colors(): Color matrix elements
  - add_background_to_entries(): Add background to matrix entries

Examples demonstrate creating matrices with different bracket types, coloring entries/rows/columns, and adding background rectangles, showcasing the class's flexibility in mathematical visualization.
