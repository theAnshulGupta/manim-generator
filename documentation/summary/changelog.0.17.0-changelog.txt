Version 0.17.0 of Manim introduced significant changes, including a complete rework of SVGMobject using the svgelements library, added support for Python 3.11 while dropping support for Python 3.7, and several notable enhancements. New features include the RemoveTextLetterByLetter animation, LineJointTypes for renderers, and RendererType enum entries. Improvements span code completion, networkx 3D layouts, and various bug fixes. The release involved contributions from 32 people, with 63 pull requests merged, focusing on SVG parsing, renderer capabilities, and overall library stability.
